Este projeto foi desenvolvido como atividade prática da disciplina de Fundamentos de Internet das Coisas, do curso Análise e Desenvolvimentos de Sistemas da PUCPR.
Com o objetivo principal de ser um sistema de estufa inteligente, monitorando a umidade e temperatura, gerando alertas caso ultrapasse os limites.

Desenvolvi este projeto utilizando a IDE: Thonny.
Sistema Operacional: Ubuntu(dentro de uma VirtualBox)
Plataforma IOT: ThingSpeake.
Hardware físico: ESP32Wroom, Relé 5v e o DTH11.

Demonstração do Projeto: https://youtu.be/RFZ9Ik42SGI.

Status: Projeto concluído e utilizado somente para fins acadêmicos.
